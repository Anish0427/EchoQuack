# **App Name**: EchoQuack

## Core Features:

- Single-Action Alert: A prominent, center-screen button that instantly triggers a push event to the paired user's device.
- FCM Direct Integration: Utilizes Firebase Cloud Messaging (FCM) via a service worker to deliver 'Quack' notifications to static web app clients.
- AI Sonic Variator: A tool that uses generative logic to subtly alter the pitch and resonance of the 'Quack' sound effect, ensuring the notification stays fresh.
- Sync Audio Payload: Ensures the specific 'Quack' audio file is pre-loaded and plays immediately upon notification arrival, even when the browser is backgrounded.
- Direct Device Binding: A streamlined setup page for two people to exchange push tokens without a traditional database or login flow.
- Persistent 'Quack' View: A minimalist, static dashboard displaying the current status of the connection and a log-less receipt indicator.

## Style Guidelines:

- Primary color: Sophisticated Plum (#6B4C85) reflecting intimate, reliable connection.
- Background color: Mist White (#F8F7F9), a desaturated variant of the primary hue for a clean, boutique agency feel.
- Accent color: Calm Slate Blue (#4C5985) used for button interactions and status indicators.
- Body and headline font: 'Poppins' for a precise, geometric, and modern look that emphasizes the app's utility and speed.
- Minimalist, thin-line vector icons with rounded terminals to maintain a friendly but high-end aesthetic.
- Centered, single-column mobile layout with massive whitespace to emphasize the core interaction button.
- Soft ripple effects (radial expands) when the notification is sent and a subtle pulse on the receiving end's interface.
